Основные допущения
===========================================
1. Программа будет проверять правильность описания структуры только после нахождения
ключевого слова struct. Неправильное написание ключевого слова обрабатываться не будет.
2.Программа не рассматривает контекст обьявления структуры, только правильность обьявления.
3. Базовыми типами считаются только "int", "char", "float" и "double". 
Модификаторы типов по типу long являются дополнением к базовому типу, а потому рассматриваться не будут.


Небольшие допущения
===========================================
3. При использовании указателя, * будет прикреплена к типу переменной:
int *name; -- плохо
int * name; -- плохо
int* name; -- хорошо
